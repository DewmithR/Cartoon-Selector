# Cartoon Selector Project

This repository contains a web-based character selector for various cartoon series. This project was created as part of a web development revision assignment.

## Project Structure
* **Garfield/**: Contains the HTML, CSS, and JS for the Garfield character selector.
* **Tom & Jerry/**: Contains a similar selector for the Tom & Jerry universe.

## Technical Requirements
* Fill a selection box with character names via JavaScript.
* Dynamically update an image display based on the selected character.
* Images are sourced from a local `/images` subfolder in JPG format.
