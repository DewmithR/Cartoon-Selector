# Garfield Character Selector

This sub-project implements a dynamic selection tool for characters from the Garfield universe.

## Characters Included
* Garfield
* Odie
* Nermal
* Pooky
* Jon

## Implementation Details
The JavaScript code populates the `<select>` element and uses an event listener to change the `src` attribute of the character image. All images are saved as `charactername.jpg` inside the `images/` directory.
