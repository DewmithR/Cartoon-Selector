# Tom & Jerry Character Selector

This sub-project implements a dynamic selection tool for characters from the Tom & Jerry universe, following the same technical requirements as the Garfield implementation.

## Characters Included
* Tom
* Jerry
* Spike
* Tyke
* Tuffy

## Technical Implementation
The JavaScript code dynamically populates the selection box with these five choices. When a character is selected, the image updates to display the corresponding `.jpg` file from the `images/` subfolder.
